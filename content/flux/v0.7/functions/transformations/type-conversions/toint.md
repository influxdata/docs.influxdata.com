The `toInt()` function converts a value to an integer.

_**Function type:** Type conversion_  
_**Output data type:** Integer_

```js
toInt()
```

## Examples
```js
from(bucket: "telegraf")
  |> filter(fn:(r) =>
    r._measurement == "mem" AND
    r._field == "used"
  )
  |> toInt()
```

## Function definition
```js
toInt = (tables=<-) =>
  tables
    |> map(fn:(r) => int(v: r._value))
```
